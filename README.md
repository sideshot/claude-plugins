# sideshot-tools

A collection of Claude Code plugins.

## Installation

Add this marketplace to Claude Code:

```bash
/plugin marketplace add sideshot/claude-plugins
```

Then install plugins:

```bash
/plugin install product-studio@sideshot-tools
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [product-studio](./plugins/product-studio) | AI-powered product image generation for e-commerce |

## Updating

To get the latest plugin updates:

```bash
/plugin marketplace update
```
