# Prompt Patterns

Effective patterns for the `--style-instructions` parameter.

## Technical Diagram
Best for installation guides and exploded views.

```text
Technical diagram style, white background, clean lines, labeled components, numbered callouts, isometric view
```

## Marketing / Hero Shot
Best for product listings and banners.

```text
Professional studio photography, soft lighting, slight drop shadow, clean white background, high detail, 8k resolution, photorealistic
```

## Infographic / Comparison
Best for "Face Frame vs Frameless" or size comparisons.

```text
Clean infographic style, side-by-side comparison, labeled sections, distinct color separation, brand color accents, Inter font labels
```

## Cross-Section / Mechanism
Best for slides, hinges, and complex hardware.

```text
Technical cross-section diagram, internal mechanism visible, engineering style, dimension lines, white background
```

## Common Modifiers

| Effect | Phrase |
|--------|--------|
| **Branding** | `"Brand color accents"`, `"Company style guide colors"` |
| **Cleanliness** | `"No text unless labeled"`, `"Clean white background"` |
| **Detail** | `"Exploded view"`, `"Internal mechanism"` |
| **Viewpoint** | `"Isometric view"`, `"Top-down plan view"`, `"Side elevation"` |
